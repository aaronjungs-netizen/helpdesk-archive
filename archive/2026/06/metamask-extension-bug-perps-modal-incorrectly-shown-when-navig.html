Symptom

  • When navigating back from the Market Details page, the Metamask extension incorrectly displays the Perps modal, causing an unexpected user experience.

Technical Root Cause

  • The error occurs due to a mismatch in the state management of the Metamask extension, specifically when handling navigation events between different pages.

Manual Overrides

  • To resolve the issue, try the following manual overrides:
  • Clear the browser's local storage to remove any cached state: localStorage.clear()
  • Reset the Metamask extension's configuration to its default settings
  • Update the web3 provider configuration using the following code block:
{
  "provider": "https://mainnet.infura.io/v3/YOUR_PROJECT_ID",
  "chainId": 1
}

Consider utilizing an external console for further debugging and configuration adjustments.

Prerequisites

  • Metamask extension installed and configured
  • A compatible Ethereum wallet set up
  • The latest version of the browser being used
