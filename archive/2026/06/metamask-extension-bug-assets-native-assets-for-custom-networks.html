Metamask Extension Bug Assets Native Assets For Custom Networks

Symptom

  • Users are unable to view native assets for custom networks when selecting them in the Metamask extension, resulting in an incomplete asset list.

Technical Root Cause

  • The error occurs due to a bug in the Metamask extension that fails to properly fetch and display native assets for custom networks, likely caused by an incorrect configuration or outdated extension version.

Manual Overrides

  • To resolve the issue, users can try the following manual overrides:
  • Update the Metamask extension to the latest version
  • Reset the extension's storage using the localStorage.clear() command
  • Manually configure the custom network's native asset using the following JSON configuration:
{
  "network": "custom-network",
  "nativeAsset": {
    "symbol": "CUSTOM",
    "name": "Custom Network Token",
    "decimals": 18
  }
}

Consider utilizing an external console for further debugging and customization.

Prerequisites

  • Custom network configured in the Metamask extension
