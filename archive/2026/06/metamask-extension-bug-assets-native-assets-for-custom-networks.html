Symptom

Technical Root Cause

  • The issue arises due to a bug in the Metamask extension that prevents it from properly retrieving and displaying native assets for custom networks, likely caused by an incorrect configuration or outdated software.

Manual Overrides

  • To resolve this issue, users can attempt the following manual overrides:

  • Add custom network configurations manually using the following JSON code block:
{

    "name": "Custom Coin",

    "decimals": 18
  }
}
