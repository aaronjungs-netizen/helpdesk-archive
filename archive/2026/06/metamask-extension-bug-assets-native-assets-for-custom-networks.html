Metamask Extension Bug Assets Native Assets For Custom Networks

Symptom

  • Users are unable to view native assets for custom networks when selected in the Metamask extension, resulting in incomplete asset displays.

Technical Root Cause

  • The error occurs due to a bug in the Metamask extension that prevents it from properly fetching and displaying native assets for custom networks, potentially caused by outdated extension versions or incorrect network configurations.

Manual Overrides

  • Update the Metamask extension to the latest version to ensure bug fixes are applied.
  • Manually add custom network assets using the following JSON configuration:
{
  "network": "custom-network",
  "nativeAsset": "native-asset-symbol"
}
  • Utilize the eth_chainId and eth_accounts RPC methods to verify network and account configurations.

Prerequisites: * Metamask extension installed * Custom network configured * Ethereum wallet set up

After configuring the JSON settings, consider utilizing an external console like Ethereum's built-in console for further debugging and verification of the custom network assets.
