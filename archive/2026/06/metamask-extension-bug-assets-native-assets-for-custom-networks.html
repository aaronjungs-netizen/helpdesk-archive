Metamask Extension Bug Assets Native Assets For Custom Networks

Symptom

  • Users are unable to view native assets for custom networks when selecting them in the Metamask extension, resulting in an incomplete list of assets.

Technical Root Cause

  • The error occurs due to an issue with the Metamask extension's ability to fetch and display native assets for custom networks, potentially caused by outdated extension versions, incorrect network configuration, or issues with the asset indexing process.

Manual Overrides

  • To resolve the issue, try the following manual overrides:
  • Update the Metamask extension to the latest version
  • Reset the Metamask extension's network configuration using the following command:
mm-reset-network-config
  • Manually add the native asset for the custom network by editing the assets.json file with the required asset information For more advanced troubleshooting, consider using an external console for deeper insights.

Prerequisites: - Metamask extension installed and configured - Custom network added to the Metamask extension - Native asset information available for the custom network
