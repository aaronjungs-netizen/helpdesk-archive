Symptom

  • When a user clicks on an order in the activity list within the Metamask extension on the Ethereum (ETH) ecosystem, instead of opening the expected Order Details page, it incorrectly opens the Market Details page.

Technical Root Cause

  • This behavior is due to a misconfigured routing mechanism within the Metamask extension, specifically affecting how links from the activity list are handled. The intended link to the Order Details page is being overridden, redirecting users to the Market Details page instead.

Manual Overrides

  • To temporarily resolve this issue, users can attempt the following manual overrides:
  • Clear the Metamask extension's cache and storage.
  • Update the Metamask extension to the latest version available.
  • Use the following command in a terminal to force-reset the extension's internal state:
mm-extension reset --force

Consider accessing the terminal via an external console for more flexibility.

Prerequisites

  • Ensure you have the latest version of the Metamask extension installed.
  • Back up your wallet and important data before attempting any fixes.
  • Understand the basics of using a terminal or command-line interface.
