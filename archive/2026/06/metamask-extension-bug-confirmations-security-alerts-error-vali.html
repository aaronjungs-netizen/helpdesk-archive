Technical Root Cause

  • The error occurs due to a mismatch or corruption in the wallet's configuration files, often resulting from an incomplete or faulty update of the Metamask extension, leading to issues with the PPOM (Payment Protocol Output Manager) module.

Manual Overrides

  • Remove and re-add the wallet to Metamask to reset configurations.
  • Update Metamask to the latest version to ensure bug fixes are applied.
