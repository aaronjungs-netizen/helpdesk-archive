Metamask Extension Bug Swap And Bridge Fail On Arbitrum Network

Symptom

  • Users experience a failure when attempting to swap or bridge assets on the Arbitrum network using the Metamask extension, resulting in an error message indicating the transaction cannot be completed.

Technical Root Cause

  • The error occurs due to a compatibility issue between the Metamask extension and the Arbitrum network, specifically related to the handling of gas prices and transaction sequencing.

Manual Overrides

  • To resolve the issue, users can try the following manual overrides:
  • Increase the gas limit and gas price for the transaction
  • Use a different RPC endpoint for the Arbitrum network
  • Update the Metamask extension to the latest version and restart the browser
