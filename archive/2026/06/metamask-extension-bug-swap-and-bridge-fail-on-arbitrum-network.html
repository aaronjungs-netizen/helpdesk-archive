Symptom

Technical Root Cause

  • The error occurs due to a compatibility issue between the Metamask extension and the Arbitrum network, specifically related to the handling of layer 2 transactions and the bridge protocol.

Manual Overrides

  • Update the Metamask extension to the latest version to ensure compatibility with the Arbitrum network.
  • Configure the Arbitrum network settings manually in the Metamask extension to ensure correct RPC and bridge settings.

  "rpc": "https://arb1.arbitrum.io/rpc",
  "bridge": "https://bridge.arbitrum.io",
  "chainId": 42161
}
