Metamask Extension Bug Swap And Bridge Fail On Arbitrum Network

Symptom

  • Users experience a failure when attempting to swap or bridge assets on the Arbitrum network using the Metamask extension, resulting in an error message indicating that the operation cannot be completed.

Technical Root Cause

  • The error occurs due to a compatibility issue between the Metamask extension and the Arbitrum network, likely caused by an outdated extension version or incorrect network configuration.

Manual Overrides

  • Update the Metamask extension to the latest version and restart the browser.
  • Configure the Arbitrum network manually by adding the following network details:
{
  "chainId": 42161,
  "chainName": "Arbitrum",
  "rpcUrls": ["https://arb1.arbitrum.io/rpc"],
  "nativeCurrency": {
    "name": "Ether",
    "symbol": "ETH",
    "decimals": 18
  }
}
  • Clear the browser cache and cookies to ensure a clean slate for the updated extension.

Prerequisites: * Metamask extension installed * Arbitrum network configured * Latest browser version
