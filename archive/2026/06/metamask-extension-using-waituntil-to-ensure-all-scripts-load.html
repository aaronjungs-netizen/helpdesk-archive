Metamask Extension Using Waituntil To Ensure All Scripts Load

Symptom

  • Users experience delayed or failed loading of web pages when using the Metamask extension, resulting in incomplete functionality or errors.

Technical Root Cause

  • The Metamask extension relies on the waitUntil function to ensure that all necessary scripts are loaded before allowing the webpage to proceed, but this function can sometimes cause issues if not implemented correctly.

Manual Overrides

  • To resolve this issue, try the following manual overrides:
  • Disable and re-enable the Metamask extension to restart the loading process.
  • Clear the browser cache and cookies to remove any corrupted data.
  • Use the following code to manually wait for the scripts to load:
{
  "content_scripts": [
    {
      "matches": ["*://*/*"],
      "js": ["contentScript.js"],
      "run_at": "document_end"
    }
  ]
}

Consider using an external console to execute the above commands for more precise control.
