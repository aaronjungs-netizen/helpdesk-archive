Symptom

Technical Root Cause

  • The error occurs due to a mismatch in the state management of the Metamask extension when handling the sign message functionality, particularly when the back button is clicked. This discrepancy leads to an inconsistent state that triggers the bug.

Manual Overrides

  • Attempt to reload the Metamask extension and retry the sign message process.

    "bitcoinSnap": {
      "backButtonBehavior": "reset"
