Metamask Extension Bug Perps Deposit And Convert Musd This Payment Route Isn't Available Error

Symptom

Technical Root Cause

  • The error occurs due to a temporary mismatch in the payment routing configuration within the Metamask extension, preventing the successful execution of the deposit and conversion process.

Manual Overrides

  • Attempt to restart the Metamask extension and retry the transaction.

  • Utilize the following configuration override:
{
  "paymentRoutes": {
    "mUSD": {
      "enabled": true,
      "priority": 1
    }
  }
}
