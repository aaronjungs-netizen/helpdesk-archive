Tronlink Bandwidth Error

Symptom

  • Users encounter an error message indicating a bandwidth issue when attempting to perform transactions or interact with smart contracts on the TRON network using TronLink.

Technical Root Cause

  • The error occurs due to insufficient bandwidth or energy on the user's TRON account, which is required to execute transactions or deploy contracts on the network.

Manual Overrides

  • Increase the account's bandwidth by freezing TRX tokens for a specified period.
  • Adjust the transaction's bandwidth and energy limits in the TronLink settings.
  • Utilize the following JSON configuration to manually set the bandwidth parameters:
{
  "bandwidth": {
    "available": 10000,
    "frozen": 5000
  },
  "energy": {
    "available": 1000,
    "frozen": 500
  }
}

Consider using an external console for more advanced configuration options.

Prerequisites

  • A TRON account with a sufficient balance of TRX tokens.
  • The latest version of TronLink installed and configured.
  • A basic understanding of TRON's bandwidth and energy mechanisms.
