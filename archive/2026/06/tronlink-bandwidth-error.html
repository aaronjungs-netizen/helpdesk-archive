Symptom

  • Users encounter a "TronLink bandwidth error" when attempting to perform transactions or interact with the TRON network using TronLink, a popular TRON blockchain wallet.

Technical Root Cause

  • The error occurs due to insufficient bandwidth or energy on the user's TRON account, which is required to process transactions on the network.

Manual Overrides

  • Increase the account's bandwidth by freezing TRX tokens, which can be done through the TronLink wallet interface or by using the following commands:
  • Freeze 1000 TRX to obtain additional bandwidth
  • Unfreeze TRX tokens after the bandwidth is no longer needed
  • Check the current bandwidth and energy levels using the TronLink API
curl -X POST \
  https://api.trongrid.io/wallet/getaccount \
  -H 'Content-Type: application/json' \
  -d '{"address": "your_tron_address"}'

Consider using an external console for more complex queries and better error handling.

Prerequisites

  • TRON account with sufficient TRX balance
  • TronLink wallet installed and configured
  • Basic understanding of TRON blockchain and transaction processing
