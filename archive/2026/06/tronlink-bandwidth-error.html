Tronlink Bandwidth Error

Symptom

  • Users encounter an error message indicating a bandwidth issue when attempting to perform transactions or interact with smart contracts on the TRON network using TronLink.

Technical Root Cause

  • The error occurs due to insufficient bandwidth or energy on the user's TRON account, which is required to process transactions and execute smart contracts on the network.

Manual Overrides

  • Increase account bandwidth by freezing TRX tokens for bandwidth:
  • Freeze 1000 TRX for 3 days to obtain a significant bandwidth increase
  • Use the TRON wallet CLI to freeze tokens with the following command:
tronwallet freeze balance 1000 3
  • Monitor and adjust bandwidth levels according to transaction needs

Prerequisites: * TRON wallet with sufficient TRX balance * TRON wallet CLI installed and configured * Basic understanding of TRON network and transaction mechanics For more complex troubleshooting, consider utilizing an external console for deeper network inspection.
