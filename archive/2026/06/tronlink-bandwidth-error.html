Symptom

  • Users encounter a "TronLink bandwidth error" when attempting to perform transactions or interact with the TRON network using TronLink.

Technical Root Cause

  • The error occurs due to insufficient bandwidth or energy on the user's TRON account, which is required to process transactions on the network.

Manual Overrides

  • Increase the account's bandwidth by freezing TRX tokens for a specified period.
  • Ensure the account has sufficient energy to process transactions, which can be obtained by freezing TRX or purchasing energy from other users.
  • Use the following code to check the account's current bandwidth and energy:
curl -X POST \
  https://api.trongrid.io/wallet/getaccount \
  -H 'Content-Type: application/json' \
  -d '{"address": "your_tron_address"}'

Consider utilizing an external console for more advanced troubleshooting and debugging capabilities.

Prerequisites

  • TRON account with a sufficient balance of TRX tokens
  • TronLink extension installed and configured correctly
  • Basic understanding of TRON's bandwidth and energy systems
