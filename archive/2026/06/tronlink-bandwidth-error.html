Tronlink Bandwidth Error

Symptom

  • Users encounter an error message indicating a bandwidth issue while attempting to perform transactions or interact with smart contracts on the TRON network using TronLink.

Technical Root Cause

  • The error occurs due to insufficient bandwidth or energy on the user's TRON account, which is required to execute transactions on the network.

Manual Overrides

  • Increase the account's bandwidth by freezing TRX for bandwidth.
  • Ensure the account has sufficient energy for the transaction by freezing TRX for energy.
  • Adjust the transaction's bandwidth and energy consumption settings:
tronweb.transactionBuilder.freezeBalance(
  '1000000', // bandwidth
  '3', // duration
  '0' // resource
);

Prerequisites: * TRON account with sufficient TRX balance * TronLink extension installed and configured * Basic understanding of TRON network resources (bandwidth and energy) After executing the above command, consider utilizing an external console for further debugging and fine-tuning of transaction parameters.
