Tronlink Bandwidth Error

Symptom

  • Users encounter an error message indicating a TronLink bandwidth issue, preventing them from completing transactions or interacting with the TRON network.

Technical Root Cause

  • The error occurs due to insufficient bandwidth on the user's TRON account, which is required to perform transactions and execute smart contracts on the network.

Manual Overrides

  • Increase the account's bandwidth by setting a higher bandwidth limit using the tronweb API.
  • Use the following JSON configuration to update the bandwidth parameters:
{
  "bandwidthLimit": 10000
}
  • Execute a transaction to claim unused bandwidth from a previous block, using a command such as tronweb.trx.freezeBalance().

Prerequisites: * TRON account with a minimum balance of 1 TRX * TronLink wallet extension installed and configured * TRON network connection established

Consider utilizing an external console, such as TronBox, to execute these commands and troubleshoot the bandwidth issue.
