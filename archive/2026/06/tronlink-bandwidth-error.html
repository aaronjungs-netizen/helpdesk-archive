TronLink Bandwidth Error

Symptom

  • Users encounter an error message indicating a bandwidth issue when attempting to perform transactions or interact with smart contracts on the TRON network using TronLink.

Technical Root Cause

  • The error occurs due to insufficient bandwidth or energy on the user's TRON account, which is required to process transactions and execute smart contract functions on the network.

Manual Overrides

  • Increase the account's bandwidth by freezing TRX tokens for bandwidth.
  • Adjust the transaction parameters to consume less bandwidth.
  • Use the following JSON-RPC method to check the account's current bandwidth:
{
  "jsonrpc": "2.0",
  "method": "getaccount",
  "params": [
    {
      "address": "your_tron_address"
    }
  ],
  "id": 1
}

Consider utilizing an external console for more complex troubleshooting and customization.

Prerequisites

  • A TRON account with sufficient TRX balance.
  • The latest version of TronLink installed.
  • A basic understanding of TRON's bandwidth and energy system.
