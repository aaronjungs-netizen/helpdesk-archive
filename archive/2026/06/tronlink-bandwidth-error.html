Tronlink Bandwidth Error

Symptom

  • Users encounter an error message indicating a bandwidth issue when attempting to perform transactions or interact with smart contracts on the TRON network using TronLink.

Technical Root Cause

  • The error occurs due to insufficient bandwidth, which is a resource required for transactions on the TRON network. Each account has a limited amount of bandwidth that can be used within a specific time frame.

Manual Overrides

  • To resolve the bandwidth error, users can try the following manual overrides:
  • Increase the account's bandwidth by freezing TRX tokens for bandwidth
  • Use a different account with sufficient bandwidth
  • Wait for the bandwidth to replenish over time
tronweb.transactionBuilder.freezeBalance(
  "your_account_address",
  "your_private_key",
  10000,
  "days",
  true
)

Prerequisites: * TronLink wallet installed and set up * Sufficient TRX tokens in the account for freezing * Basic understanding of TRON network resources (TRX, bandwidth, energy) Consider using an external console like TronWeb for more advanced troubleshooting and customization options.
