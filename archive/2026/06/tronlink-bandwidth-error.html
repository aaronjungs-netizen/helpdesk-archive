Tronlink Bandwidth Error

Symptom

  • Users encounter an error message indicating a bandwidth issue when attempting to perform transactions or interact with smart contracts on the TRON network using TronLink.

Technical Root Cause

  • The error occurs due to insufficient bandwidth or energy on the user's TRON account, which is required to execute transactions and interact with smart contracts on the network.

Manual Overrides

  • Increase the account's bandwidth by freezing TRX tokens for a specified period.
  • Adjust the transaction's bandwidth and energy limits manually to optimize resource utilization.
  • Utilize the following JSON configuration to update the TronLink settings:
{
  "bandwidthLimit": 10000,
  "energyLimit": 100000
}

Consider utilizing an external console for more advanced troubleshooting and configuration adjustments.

Prerequisites

  • TRON account with sufficient TRX balance
  • TronLink extension installed and configured
  • Basic understanding of TRON network resources (bandwidth and energy)
