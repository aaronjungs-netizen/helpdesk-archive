Symptom

  • Users encounter a "TronLink bandwidth error" when attempting to perform transactions or interact with the TRON ecosystem using the TronLink wallet.

Technical Root Cause

  • The error occurs due to insufficient bandwidth or energy on the user's account, which is required to process transactions on the TRON network. Bandwidth is a resource that is depleted with each transaction and replenished over time.

Manual Overrides

  • Increase the account's bandwidth by freezing TRX tokens for a specified period.
  • Adjust the transaction parameters, such as the transaction size or gas limit, to reduce the bandwidth consumption.
  • Utilize the following JSON configuration to manually override the bandwidth settings:
{
  "bandwidth": {
    "freeze": {
      "amount": 1000,
      "duration": 3
    }
  }
}

Consider using an external console for more advanced debugging and configuration options.

Prerequisites

  • A TronLink wallet with a TRX balance
  • A basic understanding of the TRON ecosystem and its resources (bandwidth and energy)
  • The latest version of the TronLink wallet installed and configured correctly
