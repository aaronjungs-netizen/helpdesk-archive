TronLink Bandwidth Error

Symptom

  • Users encounter an error message stating "TronLink bandwidth error" when attempting to perform transactions or interact with smart contracts on the TRON network using TronLink.

Technical Root Cause

  • The error occurs due to insufficient bandwidth or incorrect bandwidth settings in the user's TronLink account, which is required for transaction processing on the TRON network.

Manual Overrides

  • Increase the account bandwidth by freezing TRX tokens for bandwidth:
  • Use the TronLink interface to freeze TRX for bandwidth
  • Utilize the TRON command-line tools to adjust bandwidth settings
  • Adjust the bandwidth parameter in the following JSON configuration:
{
  "account": {
    "bandwidth": {
      "frozen": 1000
    }
  }
}

Consider using an external console for more complex adjustments.

Prerequisites

  • TRX tokens available for freezing
  • TronLink account set up and configured
  • Basic understanding of TRON network and TronLink functionality
