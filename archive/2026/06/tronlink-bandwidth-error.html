Tronlink Bandwidth Error

Symptom

  • Users encounter an error message indicating a TronLink bandwidth issue, preventing them from completing transactions on the TRON network.

Technical Root Cause

  • The error occurs due to insufficient bandwidth, which is a resource required for transaction processing on the TRON network, or issues with the TronLink wallet's configuration.

Manual Overrides

  • Increase the bandwidth by freezing TRX tokens:
  • Use the TronLink wallet to freeze a certain amount of TRX for bandwidth
  • Configure the wallet settings to optimize bandwidth usage
  • Utilize the following JSON configuration to update the TronLink settings:
{
  "bandwidth": {
    "frozenBalance": 1000,
    "frozenDuration": 3
  }
}

Consider utilizing an external console for more advanced troubleshooting and configuration adjustments.

Prerequisites

  • TRON network account with sufficient TRX balance
  • TronLink wallet installed and configured
  • Basic understanding of TRON network resources (TRX, bandwidth, and energy)
