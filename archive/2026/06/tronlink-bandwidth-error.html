Tronlink Bandwidth Error

Symptom

  • Users encounter an error message indicating a bandwidth issue when attempting to perform transactions or interact with smart contracts on the TRON network using TronLink.

Technical Root Cause

  • The error occurs due to insufficient bandwidth on the user's TRON account, which is required to process transactions and execute smart contract functions on the network.

Manual Overrides

  • Increase the account bandwidth by freezing TRX tokens for bandwidth.
  • Configure the TronLink wallet to automatically manage bandwidth for transactions.
  • Use the following JSON-RPC method to check the remaining bandwidth:
{
  "jsonrpc": "2.0",
  "method": "getaccount",
  "params": [
    {
      "address": "your_tron_account_address"
    }
  ],
  "id": 1
}

Consider utilizing an external console for more complex queries and better error handling.

Prerequisites

  • A TRON account with sufficient TRX balance.
  • The latest version of TronLink installed.
  • A basic understanding of TRON network concepts, including bandwidth and energy.
