Tronlink Bandwidth Error

Symptom

  • Users encounter an error message indicating a bandwidth issue when attempting to perform transactions or interact with smart contracts on the TRON network using TronLink.

Technical Root Cause

  • The error occurs due to insufficient bandwidth or energy on the user's TRON account, which is required to process transactions and execute smart contract functions.

Manual Overrides

  • Increase the account's bandwidth by freezing TRX tokens for a specified period.
  • Adjust the transaction parameters to reduce the bandwidth consumption.
  • Use the following JSON-RPC method to check the account's bandwidth:
{
  "jsonrpc": "2.0",
  "method": "tron_getAccount",
  "params": [
    {
      "address": "your_tron_address"
    }
  ],
  "id": 1
}

Consider utilizing an external console for advanced troubleshooting and transaction handling.

Prerequisites

  • TRON account with sufficient TRX balance
  • TronLink wallet extension installed and configured
  • Basic understanding of TRON network and transaction mechanics
