TronLink Bandwidth Error

Symptom

  • Users encounter an error message "TronLink bandwidth error" when attempting to perform transactions or interact with smart contracts on the TRON network using TronLink.

Technical Root Cause

  • The error occurs due to insufficient bandwidth or when the account's bandwidth is depleted, preventing further transactions until the bandwidth is replenished or the account is recharged.

Manual Overrides

  • Increase account bandwidth by calling the freeze function to lock TRX tokens, which in turn increases the available bandwidth:
tronweb.trx.freeze(tronweb.defaultAddress.base58, 10000)
  • Alternatively, recharge the account by transferring additional TRX to the address.
  • Utilize a bandwidth provider or a dApp that offers free or sponsored bandwidth for its users.

Prerequisites: * TronLink wallet installed and set up * TRX tokens available for bandwidth increase or account recharge * Basic understanding of TRON network and TronLink functionality

Consider utilizing an external console for more complex manual overrides and custom scripting.
