TronLink Bandwidth Error

Symptom

  • Users encounter an error message indicating a TronLink bandwidth issue, preventing them from completing transactions or interacting with the TRON network.

Technical Root Cause

  • The error occurs due to insufficient bandwidth or an unstable connection, causing TronLink to fail in processing requests.

Manual Overrides

  • Increase the bandwidth allocation for the user's account by using the freeze function to lock in TRX for bandwidth.
  • Clear the TronLink cache and restart the application to ensure a fresh connection.
  • Adjust the network configuration to optimize connection stability, as shown in the following example:
{
  "fullNode": "https://api.trongrid.io",
  "solidityNode": "https://api.trongrid.io",
  "eventServer": "https://api.trongrid.io"
}

For advanced troubleshooting, consider using an external console for more detailed network analysis.

Prerequisites

  • A TRON-based wallet with a sufficient TRX balance
  • A stable internet connection
  • The latest version of TronLink installed and updated
