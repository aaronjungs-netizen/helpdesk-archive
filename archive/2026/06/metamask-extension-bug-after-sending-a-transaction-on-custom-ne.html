Metamask Extension Bug After Sending A Transaction On Custom Network

Symptom

  • The user observes that their balance disappears from the Metamask extension after sending a transaction on a custom network, despite the transaction being successfully processed on the blockchain.

Technical Root Cause

  • The issue arises due to a bug in the Metamask extension that fails to properly update the balance after transactions on custom networks, potentially due to incorrect network configuration or inadequate handling of custom network transactions.

Manual Overrides

  • To troubleshoot and potentially resolve the issue, the following manual overrides can be attempted:
  • Check the custom network configuration in Metamask to ensure it matches the expected settings.
  • Verify the transaction on a blockchain explorer to confirm it was processed correctly.
  • Update Metamask to the latest version and restart the browser to apply any potential fixes.
