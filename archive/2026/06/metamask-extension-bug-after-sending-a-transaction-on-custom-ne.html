Metamask Extension Bug After Sending A Transaction On Custom Network

Symptom

  • The user observes that after sending a transaction on a custom network, their balance disappears from the Metamask extension, indicating a potential issue with the extension's functionality on custom networks.

Technical Root Cause

  • The error occurs due to a compatibility issue between the Metamask extension and custom networks, where the extension fails to properly update the balance after a transaction is sent, leading to a discrepancy in the displayed balance.

Manual Overrides

  • To resolve the issue, users can attempt the following manual overrides:
