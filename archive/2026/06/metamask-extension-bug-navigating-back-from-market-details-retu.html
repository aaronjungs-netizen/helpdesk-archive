Symptom

  • Users of the Metamask extension on the Ethereum (ETH) ecosystem experience a bug where navigating back from the Market Details page redirects them to the home screen instead of the expected Explore page.

Technical Root Cause

  • The error occurs due to a misconfigured routing mechanism within the Metamask extension, causing it to lose track of the navigation history and default to the home screen when attempting to navigate back from the Market Details page.

Manual Overrides

  • Attempt to reset the Metamask extension's navigation history by clearing browsing data and reinstalling the extension.
  • Use the following command to manually override the navigation route:
mm-extension reset-navigation --route=/explore
  • Edit the extension's configuration file to set the default return route after navigating back from the Market Details page.

You may want to use an external console for further troubleshooting.

Prerequisites

  • Metamask extension installed and configured on the browser.
  • Ethereum (ETH) ecosystem selected as the primary network.
  • Latest version of the Metamask extension updated and installed.
