Tonkeeper Jetton Transfer Loop Timeout

Symptom

  • Users experience a timeout error when attempting to transfer Tonkeeper Jettons, resulting in failed transactions and an inability to complete the transfer process. The error message explicitly states "Tonkeeper Jetton transfer loop timeout," indicating a issue with the transfer loop mechanism.

Technical Root Cause

  • The error occurs due to a malfunction in the transfer loop mechanism, which is responsible for handling the transfer of Jettons between users. This malfunction can be caused by a variety of factors, including network congestion, incorrect configuration, or issues with the Tonkeeper wallet.

Manual Overrides

  • To resolve the issue, users can attempt the following manual overrides:
  • Check the Tonkeeper wallet configuration to ensure it is correctly set up and functioning as expected.
