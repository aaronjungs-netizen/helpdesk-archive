Tonkeeper Jetton Transfer Loop Timeout

Symptom

  • The user encounters a "Tonkeeper Jetton transfer loop timeout" error message when attempting to transfer jettons, indicating that the transfer process has exceeded the allowed time limit.

Technical Root Cause

  • This error occurs due to a prolonged waiting period for transaction confirmation on the TON blockchain, often resulting from network congestion or high gas fees.

Manual Overrides

  • To resolve the issue, try the following manual overrides:
  • Increase the gas limit to prioritize the transaction
  • Reduce the transfer amount to minimize transaction size
  • Utilize an alternative transfer method, such as a different wallet or exchange
