Tonkeeper Jetton Transfer Loop Timeout

Symptom

  • Users encounter a "Tonkeeper Jetton transfer loop timeout" error when attempting to transfer jettons, resulting in failed transactions and potential loss of funds. The error message typically appears after a prolonged period of processing, indicating a timeout in the transfer loop.

Technical Root Cause

  • The error occurs due to a timeout in the Tonkeeper protocol's jetton transfer mechanism, which is designed to handle the complexities of token transfers within the TON ecosystem. This timeout can be caused by various factors, including network congestion, high gas prices, or issues with the smart contract.

Manual Overrides

  • To resolve the issue, users can attempt the following manual overrides:
  • Increase the gas limit to allow for more complex transactions
  • Implement a retry mechanism with exponential backoff to handle temporary network issues
  • Update the Tonkeeper protocol to the latest version, which may include fixes for known issues
