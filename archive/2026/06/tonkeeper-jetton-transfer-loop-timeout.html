Tonkeeper Jetton Transfer Loop Timeout

Symptom

  • Users experience a timeout error when attempting to transfer Tonkeeper Jettons, resulting in failed transactions and potential loss of assets. The error message "Tonkeeper Jetton transfer loop timeout" is displayed, indicating a issue with the transfer process.

Technical Root Cause

  • The error occurs due to a prolonged delay in the transfer loop, exceeding the maximum allowed time threshold. This can be caused by network congestion, high gas fees, or issues with the Tonkeeper wallet.

Manual Overrides

  • To resolve the issue, users can try the following manual overrides:
  • Increase the gas limit to prioritize the transaction
  • Use a different wallet or transfer method to bypass potential issues with Tonkeeper
  • Implement a custom transfer script using the following code:
tonos-cli call <wallet_address> transfer "{\"dest\":\"<recipient_address>\",\"amount\":\"<amount>\"}" --abi TonkeeperWallet.abi

Prerequisites: - Tonos-cli installed and configured on the system - Tonkeeper wallet and recipient addresses available - Sufficient funds in the wallet for the transfer
