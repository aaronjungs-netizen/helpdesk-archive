Symptom

  • Users experience a "Tonkeeper Jetton transfer loop timeout" error when attempting to transfer jettons, resulting in failed transactions and an inability to proceed with their intended actions.

Technical Root Cause

  • The error occurs due to a timing issue in the Tonkeeper application, where the transfer loop exceeds the maximum allowed time, causing the transaction to timeout before completion.

Manual Overrides

  • To resolve the issue, users can try the following manual overrides:
  • Increase the timeout threshold in the Tonkeeper settings to allow for more time to complete the transaction.
  • Verify the jetton balance and ensure sufficient funds are available for the transfer.
  • Use the following command to update the Tonkeeper configuration:
tonos-cli config set timeout 300

Consider using an external console for more advanced configuration options.

Prerequisites

  • Tonkeeper application installed and configured
  • Sufficient jetton balance for the transfer
  • Stable internet connection for transaction processing
