Symptom

  • Users experience a "Tonkeeper Jetton transfer loop timeout" error when attempting to transfer jettons, indicating a failure in the transaction process that results in an infinite loop of attempts to complete the transfer.

Technical Root Cause

  • The error occurs due to a misconfiguration or overload in the TON (Telegram Open Network) ecosystem, specifically within the Tonkeeper wallet's interaction with the blockchain, leading to a timeout in the transaction loop.

Manual Overrides

  • Check the TON blockchain for network congestion and high gas fees, which can slow down transactions.
  • Verify that the Tonkeeper wallet is updated to the latest version to ensure compatibility with the current TON network state.
