Tonkeeper Jetton Transfer Loop Timeout

Symptom

  • The user experiences a timeout error when attempting to transfer Tonkeeper Jettons, resulting in a failed transaction and an error message indicating a loop timeout.

Technical Root Cause

  • The error occurs due to a temporary network congestion or a high load on the TON blockchain, causing the transfer operation to exceed the maximum allowed time, thus triggering a timeout.

Manual Overrides

  • To resolve the issue, try the following manual overrides:
  • Increase the gas limit for the transaction to prioritize it in the network
  • Reduce the number of concurrent transactions to alleviate network congestion
  • Implement a retry mechanism with exponential backoff to handle transient failures
