Tonkeeper Jetton Transfer Loop Timeout

Symptom

  • The user encounters a "Tonkeeper Jetton transfer loop timeout" error message when attempting to transfer jettons, indicating that the transfer process has exceeded the allocated time limit.

Technical Root Cause

  • This error occurs due to the inability of the Tonkeeper wallet to complete the jetton transfer within the predefined timeout period, often caused by network congestion, high gas fees, or insufficient funds.

Manual Overrides

  • To resolve the issue, try the following manual overrides:
  • Increase the gas limit for the transaction
  • Reduce the number of jettons being transferred to minimize transaction size
  • Utilize a custom configuration to bypass the default timeout threshold
{
  "timeout": 30000,
  "gasLimit": 20000,
  "jettonAmount": 10
}

Prerequisites: * Tonkeeper wallet installed and configured * Sufficient TON coins for gas fees * Stable internet connection

Consider utilizing an external console to execute these commands for more precise control over the transaction parameters.
