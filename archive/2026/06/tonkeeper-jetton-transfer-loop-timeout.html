Symptom

  • Users experience a "Tonkeeper Jetton transfer loop timeout" error when attempting to transfer jettons, resulting in failed transactions and inability to complete the transfer process.

Technical Root Cause

  • The error occurs due to a timeout in the transfer loop, which can be caused by network congestion, high gas fees, or issues with the Tonkeeper wallet configuration.

Manual Overrides

  • Increase the gas limit to prioritize the transaction and avoid timeouts:
  • Update the Tonkeeper wallet configuration to use a higher gas limit
  • Use the tonos-cli command to manually set the gas limit for the transfer transaction
  • Utilize the following code block to adjust the transfer parameters:
tonos-cli call 0:xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx setGasLimit 1000000

Consider utilizing an external console for more advanced troubleshooting and configuration adjustments.

Prerequisites

  • Tonkeeper wallet installed and configured
  • Basic understanding of TON blockchain and jetton transfer process
  • tonos-cli command-line tool installed and accessible
