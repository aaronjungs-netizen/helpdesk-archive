Tonkeeper Jetton Transfer Loop Timeout

Symptom

  • Users encounter a "Tonkeeper Jetton Transfer Loop Timeout" error message when attempting to transfer jettons, indicating a failure in the transaction process. The error is typically accompanied by a prompt to retry the transaction or check the network status.

Technical Root Cause

  • The error occurs due to a timeout in the transfer loop, which can be caused by network congestion, high transaction volumes, or issues with the Tonkeeper wallet's connection to the TON blockchain.

Manual Overrides

  • To resolve the issue, try the following manual overrides:
  • Increase the transaction timeout threshold to allow for longer processing times.
  • Use a different transaction routing path to bypass congested network segments.
  • Implement a retry mechanism with exponential backoff to handle temporary failures.
