Technical Root Cause

  • The error occurs due to a timeout in the Tonkeeper jetton transfer loop, which can be caused by network congestion, high gas prices, or issues with the Tonkeeper smart contract.

Manual Overrides

  • Increase the timeout threshold: Adjust the configuration to allow for longer transaction processing times.
  • Optimize gas prices: Lower gas prices can help reduce network congestion and decrease the likelihood of timeouts.
  • Update Tonkeeper contract: Ensure the Tonkeeper smart contract is up-to-date, as newer versions may include fixes for known issues.
