Symptom

  • The user experiences a "Tonkeeper Jetton transfer loop timeout" error when attempting to transfer jettons, resulting in failed transactions and potential loss of funds.

Technical Root Cause

  • The error occurs due to a timeout in the Tonkeeper jetton transfer loop, which can be caused by network congestion, high transaction volumes, or issues with the Tonkeeper wallet configuration.

Manual Overrides

  • To resolve the issue, try the following manual overrides:
  • Increase the transaction timeout threshold
  • Reduce the transaction amount to alleviate network congestion
  • Verify the Tonkeeper wallet configuration to ensure it is up-to-date and correctly set up
tonos-cli config set --timeout 300

Consider utilizing an external console for more advanced troubleshooting and configuration options.

Prerequisites

  • Tonos-cli installed and configured on the system
  • A basic understanding of TON blockchain and Tonkeeper wallet
  • Access to the Tonkeeper wallet configuration files
