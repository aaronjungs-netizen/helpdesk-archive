Tonkeeper Jetton Transfer Loop Timeout

Symptom

  • Users experience a timeout error when attempting to transfer Tonkeeper jettons, resulting in failed transactions and an inability to complete the transfer process.

Technical Root Cause

  • The error occurs due to a temporary mismatch between the Tonkeeper wallet's internal state and the TON blockchain, causing the transfer loop to exceed the maximum allowed time, thus timing out.

Manual Overrides

  • To resolve the issue, users can attempt the following manual overrides:
  • Restart the Tonkeeper wallet application to resynchronize with the TON blockchain.
  • Check for any pending transactions and cancel them if necessary.
  • Use the following command to manually update the wallet's internal state:
tonos-cli call <wallet_address> resizeWallet 0x0 {} --abi Wallet.abi

Consider utilizing an external console for more advanced troubleshooting and manual intervention.

Prerequisites

  • Tonkeeper wallet application installed and configured.
  • Sufficient TON balance to cover transaction fees.
  • Stable internet connection for seamless blockchain synchronization.
