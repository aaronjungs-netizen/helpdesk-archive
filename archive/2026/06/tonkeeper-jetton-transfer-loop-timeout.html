Tonkeeper Jetton Transfer Loop Timeout

Symptom

  • The user encounters a timeout error when attempting to transfer Tonkeeper Jettons, resulting in a failed transaction and an error message indicating a transfer loop timeout.

Technical Root Cause

  • The error occurs due to a temporary network congestion or a high load on the TON blockchain, causing the transfer operation to exceed the maximum allowed time, thus timing out.

Manual Overrides

  • To resolve the issue, the user can try the following manual overrides:
  • Increase the transaction timeout threshold to allow for more time to complete the transfer.
  • Reduce the number of concurrent transactions to alleviate network congestion.
  • Implement a retry mechanism with exponential backoff to handle temporary timeouts.
