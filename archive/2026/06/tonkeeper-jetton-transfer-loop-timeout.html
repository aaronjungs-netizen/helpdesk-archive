Symptom

  • The user encounters a "Tonkeeper Jetton transfer loop timeout" error when attempting to transfer jettons, indicating that the transfer process has exceeded the allotted time frame, resulting in a failed transaction.

Technical Root Cause

  • This error occurs due to a prolonged delay in the transaction confirmation process, which can be caused by network congestion, slow transaction processing, or issues with the Tonkeeper wallet's connection to the TON blockchain.

Manual Overrides

  • To resolve the issue, try the following manual overrides:
  • Increase the transaction timeout limit in the Tonkeeper wallet settings.
  • Restart the Tonkeeper wallet application to re-establish a stable connection to the TON blockchain.
  • Use the following command to manually update the transaction timeout configuration:
{
  "timeout": 30000,
  "retry_count": 5
}

Consider using an external console to execute low-level commands for more advanced troubleshooting.

Prerequisites

  • Tonkeeper wallet installed and configured on the device.
  • A basic understanding of TON blockchain transactions and wallet management.
  • Access to the Tonkeeper wallet settings and configuration options.
