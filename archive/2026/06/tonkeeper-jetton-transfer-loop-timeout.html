Tonkeeper Jetton Transfer Loop Timeout

Symptom

  • Users experience a timeout error when attempting to transfer Tonkeeper Jettons, resulting in failed transactions and an unclear status of their tokens.

Technical Root Cause

  • The error occurs due to a synchronization issue between the user's wallet and the TON blockchain, causing the transfer process to hang indefinitely.

Manual Overrides

  • To resolve the issue, users can attempt the following steps:
  • Check the wallet's connection to the TON network and ensure it is stable.
  • Restart the Tonkeeper application to reset the transfer loop.
  • Execute the following command to manually refresh the wallet's state:
tonos-cli call <wallet_address> refresh {} --abi Wallet.abi

Consider utilizing an external console for more complex troubleshooting.

Prerequisites

  • A stable connection to the TON network
  • The latest version of Tonkeeper installed
  • A basic understanding of command-line interfaces
