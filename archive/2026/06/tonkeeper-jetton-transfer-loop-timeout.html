Tonkeeper Jetton Transfer Loop Timeout

Symptom

  • Users encounter a timeout error when attempting to transfer Tonkeeper Jettons, resulting in failed transactions and potential loss of funds.

Technical Root Cause

  • The error occurs due to congestion in the TON blockchain network or issues with the Tonkeeper wallet's connection to the network, causing the transfer process to exceed the maximum allowed time limit.

Manual Overrides

  • To resolve the issue, try the following manual overrides:
  • Increase the gas limit for the transaction to prioritize it in the network queue.
  • Use a different TON proxy server to bypass potential network congestion.
  • Update the Tonkeeper wallet to the latest version to ensure compatibility with the latest TON blockchain updates.
