Tonkeeper Jetton Transfer Loop Timeout

Symptom

  • Users encounter a timeout error when attempting to transfer Tonkeeper Jettons, resulting in failed transactions and an inability to complete the transfer process.

Technical Root Cause

  • The error occurs due to a malfunction in the transfer loop mechanism, which is responsible for handling the transfer of Jettons between users. This malfunction can be caused by issues such as network congestion, outdated smart contract versions, or incorrect configuration of the transfer loop parameters.

Manual Overrides

  • To resolve the issue, users can attempt the following manual overrides:
  • Check the network configuration and ensure that it is properly connected to the TON blockchain.
  • Update the smart contract version to the latest release.
  • Adjust the transfer loop parameters to optimize the transfer process.
