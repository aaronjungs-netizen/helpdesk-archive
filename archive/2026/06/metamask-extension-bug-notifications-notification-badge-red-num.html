Metamask Extension Bug Notifications Notification Badge Red Num

Symptom

  • The Metamask extension icon displays a red notification badge with a number, indicating unread notifications, but the notifications themselves are not visible or have already been read.

Technical Root Cause

  • The error occurs due to a synchronization issue between the Metamask extension and the Ethereum blockchain, causing the notification badge to malfunction and display an incorrect number of unread notifications.

Manual Overrides

  • Check for updates: Ensure the Metamask extension is updated to the latest version.
  • Clear cache: Clear the browser's cache and cookies to resolve potential synchronization issues.
  • Reset notifications: Use the following command to reset the notification badge:
{
  "notifications": {
    "badge": 0
  }
}

Consider utilizing an external console to execute further diagnostic commands.

Prerequisites

  • Metamask extension installed and configured
  • Ethereum wallet set up and connected to the extension
  • Browser updated to the latest version
