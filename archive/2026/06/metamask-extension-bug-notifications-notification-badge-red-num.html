Metamask Extension Bug Notifications Notification Badge Red Num

Symptom

  • The Metamask extension displays a red notification badge with a number, indicating unread notifications, but the notifications are not visible or have already been read.

Technical Root Cause

  • The error occurs due to a caching issue or a miscommunication between the Metamask extension and the Ethereum network, causing the notification badge to remain even after notifications have been acknowledged.

Manual Overrides

  • Clear the Metamask extension's local storage by running the following command in the browser's console:
localStorage.clear("metaMask");
  • Disable and re-enable the Metamask extension to restart its internal state.
  • Check for any pending transactions or notifications on the Ethereum network using a blockchain explorer.

Prerequisites: * Metamask extension is installed and up-to-date * Ethereum network connection is stable * Browser cache is cleared Consider utilizing an external console for further debugging.
