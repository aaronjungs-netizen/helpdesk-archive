Metamask Extension Bug Notifications Notification Badge Red Num

Symptom

  • The Metamask extension displays a red notification badge with a number, indicating unread notifications, but the notifications are not visible or have already been read.

Technical Root Cause

  • The error occurs due to a mismatch between the local storage of notifications and the actual notification count, often caused by an incomplete or corrupted data synchronization process.

Manual Overrides

  • Clear the local storage of notifications by running the following command in the browser's console: 
localStorage.removeItem('metaMask.notifications');
  • Disable and re-enable the Metamask extension to restart the notification synchronization process.
  • Manually refresh the notifications by clicking on the Metamask icon and selecting "Refresh" from the dropdown menu.

Consider using an external console for more advanced debugging and troubleshooting.

Prerequisites

  • Metamask extension installed and enabled
  • Ethereum network selected
  • Browser cache cleared
