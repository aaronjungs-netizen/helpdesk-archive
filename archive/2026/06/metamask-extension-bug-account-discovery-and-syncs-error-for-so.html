Metamask Extension Bug Account Discovery And Syncs Error For Social Login Import Users

Symptom

  • Users encounter an error when attempting to import their account using social login, resulting in a failure to discover and sync accounts within the Metamask extension.

Technical Root Cause

  • The error occurs due to a mismatch between the expected and actual account formats during the social login import process, often caused by outdated extension versions or incorrect configuration settings.

Manual Overrides

  • To resolve the issue, try the following steps:
  • Update the Metamask extension to the latest version
  • Clear the browser cache and restart the browser
  • Configure the social login settings using the following JSON format:
{
  "socialLogin": {
    "providers": ["google", "facebook"],
    "accountId": "0x742d35Cc6634C0532925a3b844Bc454e4438f44e"
  }
}

Consider utilizing an external console for advanced troubleshooting and configuration.

Prerequisites

  • Social login providers set up and enabled
  • Latest version of the browser and operating system
