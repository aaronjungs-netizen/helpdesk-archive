Metamask Extension P0 Investigate And Cap Multichain Account Creation/Syncing Span Fan-Out

Symptom

  • Users experience issues with Metamask extension, specifically with multichain account creation and syncing, resulting in a span fan-out error, indicated by the error code (#1).

Technical Root Cause

  • The error occurs due to excessive account creation and syncing requests, causing a fan-out effect that overloads the system, leading to a P0 (Priority 0) error.

Manual Overrides

  • To resolve the issue, try the following manual overrides:
  • Set a limit on the number of accounts that can be created and synced simultaneously.
  • Implement a rate limiter to control the frequency of requests.
  • Use the following code to override the default settings:
{
  "accountCreationLimit": 5,
  "syncingLimit": 3,
  "rateLimit": 10
}
