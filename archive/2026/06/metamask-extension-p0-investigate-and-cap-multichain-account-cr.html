Metamask Extension P0 Investigate And Cap Multichain Account Creation/Syncing Span Fan-Out

Symptom

  • Users experience excessive account creation and syncing issues with Metamask extension, resulting in performance degradation and potential crashes.

Technical Root Cause

  • The error occurs due to uncontrolled multichain account creation and syncing, causing a fan-out effect that overwhelms the system.

Manual Overrides

  • Stop the Metamask extension and restart it to reset the account syncing process.
  • Limit the number of accounts created per chain using the following configuration:
{
  "accountCreationLimit": 5,
  "chainSyncLimit": 3
}
  • Use the eth_accounts API to manually retrieve and manage accounts, ensuring a more controlled approach to account creation and syncing. Consider utilizing an external console for more advanced debugging and configuration.
