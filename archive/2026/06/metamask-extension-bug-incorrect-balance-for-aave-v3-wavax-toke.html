Metamask Extension Bug Incorrect Balance For Aave V3 Wavax Token

Symptom

  • Users of the Metamask extension on the Avalanche network may notice that their Aave v3 WAVAX token balance is displayed incorrectly, potentially causing confusion and issues with transactions.

Technical Root Cause

  • The error occurs due to a bug in the Metamask extension's handling of token balances on the Aave v3 protocol, specifically affecting the WAVAX token on the Avalanche network.

Manual Overrides

  • To resolve the issue, users can try the following manual overrides:
  • Check the token balance on the Aave v3 protocol's official website or a blockchain explorer to verify the correct balance.
  • Update the Metamask extension to the latest version, as newer versions may include fixes for the bug.
  • Use the following JSON-RPC command to refresh the token balance:
{
  "jsonrpc": "2.0",
  "method": "eth_call",
  "params": [
    {
      "to": "0x...AaveV3ContractAddress...",
      "data": "0x...balanceOfMethodSignature..."
    },
    "latest"
  ],
  "id": 1
}

Consider utilizing an external console, such as a dedicated JSON-RPC client, for more advanced troubleshooting.

Prerequisites

  • Metamask extension installed and configured on the Avalanche network
  • Aave v3 protocol's WAVAX token added to the Metamask extension
  • Basic understanding of JSON-RPC commands and blockchain explorers
