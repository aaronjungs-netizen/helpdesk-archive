Metamask Extension Bug Incorrect Balance For Aave V3 Wavax Token

Symptom

  • Users of the Metamask extension may notice an incorrect balance display for their Aave v3 WAVAX tokens on the Avalanche network, potentially causing confusion and incorrect financial decisions.

Technical Root Cause

  • The issue arises from a synchronization problem between Metamask and the Aave v3 protocol on the Avalanche network, leading to a mismatch in the reported token balance.

Manual Overrides

  • Update Metamask to the latest version to ensure compatibility with the Aave v3 protocol.
  • Reset the Metamask account by removing and re-adding the Avalanche network.
  • Utilize the following JSON-RPC method to manually fetch the token balance:
{
  "jsonrpc": "2.0",
  "method": "eth_call",
  "params": [
    {
      "to": "0x...AaveV3ContractAddress...",
      "data": "0x...balanceOfMethodSignature..."
    },
    "latest"
  ],
  "id": 1
}

  • Aave v3 protocol integrated with the Avalanche network.
  • Basic understanding of JSON-RPC methods and token contract interactions.
