Metamask Extension Bug Incorrect Balance For Aave V3 Wavax Token

Symptom

  • Users of the Metamask extension on the Avalanche network may notice that their Aave v3 WAVAX token balance is displayed incorrectly, potentially causing confusion and affecting their ability to manage their assets effectively.

Technical Root Cause

  • The error occurs due to a bug in the Metamask extension's handling of token balances on the Aave v3 protocol, specifically affecting the WAVAX token on the Avalanche network. This discrepancy can arise from incorrect cache handling, outdated protocol implementations, or issues with the extension's ability to fetch the latest balance data.

Manual Overrides

  • To troubleshoot this issue, users can attempt the following manual overrides:
  • Clear the Metamask extension's cache and restart the browser.
  • Manually fetch the latest token balance by using the Aave v3 protocol's API: bash curl -X GET \ https://api.aave.com/api/v2/markets/avalanche \ -H 'Content-Type: application/json'
  • Update the Metamask extension to the latest version to ensure the bug has been patched. For more precise control, consider utilizing an external console to execute the above commands.

Prerequisites

  • Metamask extension installed and configured on the Avalanche network.
  • Aave v3 protocol supported and enabled within the Metamask extension.
  • WAVAX token added to the Metamask extension's wallet.
