Metamask Extension Bug Incorrect Balance For Aave V3 Wavax Token

Symptom

  • Users notice an incorrect balance display for the Aave v3 WAVAX token on the Avalanche network when using the Metamask extension, despite having a valid token balance.

Technical Root Cause

  • The error occurs due to a bug in the Metamask extension that fails to properly fetch and update the token balance for Aave v3 WAVAX on the Avalanche network, resulting in an incorrect display of the user's token holdings.

Manual Overrides

  • Update the Metamask extension to the latest version to ensure the bug is fixed.
  • Manually add the Aave v3 WAVAX token to the Metamask wallet using the token's contract address.
  • Use the following JSON payload to configure the token settings:
{
  "tokenAddress": "0x...WAVAX contract address...",
  "tokenSymbol": "WAVAX",
  "tokenDecimal": 18,
  "tokenImage": "https://...WAVAX token image..."
}

For advanced troubleshooting, consider using an external console to execute low-level commands.

Prerequisites

  • Metamask extension installed and configured on the browser.
  • Aave v3 WAVAX token added to the Metamask wallet.
  • Avalanche network selected as the active network in Metamask.
