Metamask Extension Bug Incorrect Balance For Aave V3 Wavax Token

Symptom

  • Users of the Metamask extension on the Avalanche network may notice that their Aave v3 WAVAX token balance is displayed incorrectly, potentially leading to confusion and incorrect transactions.

Technical Root Cause

  • The error occurs due to a bug in the Metamask extension's handling of the Aave v3 protocol on the Avalanche network, resulting in an incorrect balance calculation for the WAVAX token.

Manual Overrides

  • To resolve the issue, users can try the following manual overrides:
  • Update the Metamask extension to the latest version
  • Reset the Metamask account settings to default
  • Use the following JSON-RPC command to fetch the correct balance:
{
  "jsonrpc": "2.0",
  "method": "eth_call",
  "params": [
    {
      "to": "0x...AaveV3ContractAddress...",
      "data": "0x...balanceOfMethodSignature..."
    },
    "latest"
  ],
  "id": 1
}
