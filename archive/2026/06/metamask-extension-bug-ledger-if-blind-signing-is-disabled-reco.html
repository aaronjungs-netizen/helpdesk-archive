Metamask Extension Bug Ledger If Blind Signing Is Disabled Reco

Symptom

  • Users encounter an error when attempting to reconnect the Ledger device to Metamask, specifically after disabling blind signing, resulting in a workflow failure.

Technical Root Cause

  • The issue arises due to incompatibility between Metamask's blind signing settings and the Ledger device's configuration, causing a disruption in the connection process.

Manual Overrides

  • To resolve the issue, try the following steps:
  • Enable blind signing on the Ledger device and restart Metamask.
  • Update Metamask to the latest version to ensure compatibility with the Ledger device.
  • Run the following command to reset the Ledger connection:
mm-reset-ledger-connection --force-reconnect

Consider utilizing an external console for advanced troubleshooting and configuration adjustments.

Prerequisites

  • Metamask extension installed and updated to the latest version.
  • Ledger device properly configured and connected to the computer.
  • Blind signing enabled or disabled according to the user's preference.
