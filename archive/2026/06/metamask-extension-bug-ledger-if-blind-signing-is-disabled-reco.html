Metamask Extension Bug Ledger If Blind Signing Is Disabled Reco

Symptom

  • Users encounter an error when attempting to reconnect to the Ledger device in Metamask, specifically after disabling blind signing, resulting in a failed reconnection process.

Technical Root Cause

  • The error occurs due to a bug in the Metamask extension that fails to properly handle the reconnection workflow when blind signing is disabled on the Ledger device, leading to a timeout or disconnection.

Manual Overrides

  • To resolve the issue, try the following manual overrides:
  • Enable and then disable the Ledger device in Metamask to reset the connection.
  • Update the Metamask extension to the latest version to ensure any bug fixes are applied.
  • Use the following command to reset the Ledger connection manually:
mm-reset-ledger-connection --force

Consider utilizing an external console for advanced troubleshooting.

Prerequisites

  • Metamask extension installed and configured
  • Ledger device properly set up and connected
  • Blind signing disabled on the Ledger device
